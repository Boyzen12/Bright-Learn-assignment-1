# Bright-Learn-assignment-1
Introductory assignment covering python, JavaScript, and SQL fundamentals
This repository contains my first research assignment for the BrightLearn Data Analytics and Data Science course. The assignment focuses on foundational concepts in data analytics, including data fundamentals such as databases, data warehouses, and ETL processes, as well as key topics in artificial intelligence, machine learning, APIs, and data ethics. Through this assignment, I researched and explained core concepts using beginner-level knowledge and real-world examples to strengthen my understanding of how data is used in modern systems. This work reflects my journey as a beginner in data analytics, building a solid foundation for future learning and practical application, with a strong interest in using data to solve real-world problems.
